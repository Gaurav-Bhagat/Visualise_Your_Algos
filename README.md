# Sorting & Graph Algorithm Visualizer

An interactive web-based visualizer built using **React.js** that helps users understand how popular **sorting** and **graph algorithms** work through smooth animations and visual feedback.


### Sorting Algorithms
- Bubble Sort
- Selection Sort 
- Insertion Sort 
- Merge Sort
- Quick Sort
- Heap Sort
  
### Graph Algorithms
- **Dijkstra’s Algorithm** (Shortest Path)
- **Topological Sort** (Kahn’s Algorithm)

### ✅ UI Features
- Visual feedback with color-coded steps
- Dark mode-friendly (black background)
- give inputs of graph get your path
- Fast animation
- get the dry runs
